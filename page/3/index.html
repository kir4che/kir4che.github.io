Browser Object Model（BOM）

瀏覽器本身（window 物件）所有功能的核心，作為 JS 與瀏覽器的溝通窗口，無關網頁內容。

結構

  • screen：使用者的螢幕畫面資訊
  • location：頁面網址 (URL)
  • history：瀏覽器的上下頁
  • navigator：瀏覽器資訊
  • frames：瀏覽器框架
