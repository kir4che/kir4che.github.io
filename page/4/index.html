  • 類別 class：定義某種物件的 property

ex. Java、C++

[JS] 提升 Hosting

2022-03-07

簡介

將變數、函式及 class 於創造階段時，先行指向記憶體位置 ➪ 往上提到函式頂端

JS 執行時的一種現象（想像），未規範於 ECMA ES6。

影響

  • 不用先宣告變數，養成事後補上習慣 ➪ 可讀性降低
  • 函式宣告可擺後面
  • 互相呼叫函式

運作

執行函式後，但還沒開始跑內部程式碼之前依順序做出：

  1. 把參數放到 VO 並設定值，傳進什麼就是什麼，沒有值則設成 undefined。
  2. 把函式宣告放到 VO，若有同名則覆蓋掉。
  3. 把變數宣告（賦值不變）放到 VO，若有同名則忽略。

[JS] 作用域 Scope

2022-03-04

規範哪些變數在哪些程式碼中可以取用的規則

作用範圍

全域作用域 Global Scope

函式外宣告

  • 瀏覽器頁面開啟時，自動生成一全域性作用域 window
  • 會一直存在到頁面關閉為止

缺點

  • 允許重複宣告

  • 不支援 Block Scope

  • 不支援常數

影響

  • 命名空間汙染、混亂

  • 易被存取，可能被修改而導致臭蟲。

  • 性能差，須一直追蹤。

區域作用域 Local Scope

  • 函式作用域 Function Scope

    函式 {} 內宣告

  • 區塊作用域 Block Scope（ES6）

    於函式中的 { } 內宣告 ➪ 防止不當存取
