[JS30] Day02 CSS＋JS Clock

2022-03-06

功能

用 JS、CSS 製作一個實時的時鐘

控制時間

  • 不斷於間隔固定的時間重複執行 setInterval()
  • 執行一次就結束 setTimeout ()

Date

日期

  • 年（yyyy）getFullYear()
  • 月（0-11）getMonth()
  • 日（1-31）getDate()
  • 星期幾（0-6）getDay()

時間

  • 時（0-23）getHours()

  • 分（0-59）getMinutes()

  • 秒（0-59）getSeconds()

時鐘角度

  1. 算出比例再＊360°

  2. 時針、分針再加上其分、秒

  3. 每秒改變的角度再 +90°

  4. 時針、分針、秒針隨秒逐漸增加

    • 秒針 360／60＝6°/秒
    • 分針 360／60＝6°/分
    • 時針 360／12＝30°/時
