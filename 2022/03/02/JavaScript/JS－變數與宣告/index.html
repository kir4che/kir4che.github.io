[JS] 變數與宣告

2022-03-02

變數 Variables

變數是對值的引用，名稱被稱為識別符 identifier（具有唯一性）需要遵從一定規則。

命名規則

  • 開頭可用 ASCII（英文大小寫）或下底線（_）、錢符號（$），不可用數字。
  • 其他則可用英文、數字或下底線（_）
  • 大小寫有差異
  • 不可用 JS 保留字

類型

全域變數 Global Variable

  • 任何地方皆能存取到
  • 在 function 外宣告的變數

區域變數 Local Variable

  • 在 function 內宣告的變數
  • 有效於該作用域內
  • 執行後自動銷毀，並同步釋放記憶體。

宣告

不宣告也能使用變數，取而代之的是要指定其值（嚴格模式下報錯）

var

宣告一個可隨意改值的變數

  • 若無初始值則預設 undefined

  • 變數執行可以比宣告還早（∵ Hoisting）

  • 函式作用域

let vs const（ES6）

相同點

  • 避免汙染全域變數
  • 變數不會提升
  • 區塊作用域
  • 同區塊內禁止重複命名（會報錯）

let

宣告一個可隨意改值的區域變數

  • 一次宣告多個變數和值
  • 若無初始值則預設 undefined
  • 可重複給值

const

宣告一個只可讀取的不可變常數

  • 宣告時必須賦值
  • 不能重新給值
