[JS] 物件 Object

2022-03-10

物件導向 Object-oriented

JS 是一種基於原型，而不是基於類的物件導向語言。

  • 採用物件 Objects 來模塑真實的實物世界
  • 每一物件都有自己的屬性與方法

基於類別 Class-Based
擁有「類別」與「實例」的概念

  • 類別 class：定義某種物件的 property
  • 實例 instance：類別產生的 object

ex. Java、C++

基於原型 Prototype-Based
沒有類別與實體的概念，只有物件，且新物件在初始化時以原型物件為範本獲得屬性。

ex. JavaScript

特性

  • 封裝 Encapsulation
  • 繼承 Inheritance
  • 多型 Polymorphism

實體化 Instantiation

類別 Class -> 建構子 Constructor -> 物件實體 Object instance

  • class：建構某特定物體的設計圖
  • instance：按照設計圖所製造的成品
  • constructor：初始化的動作
