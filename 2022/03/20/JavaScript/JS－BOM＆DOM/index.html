瀏覽器物件模型 Browser Object Model（BOM）

瀏覽器所有功能的核心（window 物件），JS 與瀏覽器溝通的窗口，無關網頁內容。

window

ECMAScript 標準裡的全域物件，代表瀏覽器視窗本身。

💡若視窗中含有框架，則每一框架都有屬於自己的 window 物件。

結構

  • screen 螢幕畫面尺寸
  • location 頁面網址
  • history 上下頁切換（不建議使用）
  • navigator 瀏覽器資訊
  • frames 框架

常用操作

  • window.open() 開啟新視窗 ↔ window.close()

  • window.print()

  • alert() 警告

  • confirm() 確認

  • prompt() 文字輸入

  • setTimeout() 執行一次就停止 ↔ clearTimeout()

  • ``setInterval()一直執行下去 ↔clearInterval()`

文件物件模型 Document Object Model（DOM）

JS 控制網頁的節點與內容的標準，將文件內的各個標籤（文字、圖片等）定義成物件後，最終皆會形成一樹狀結構。

DOM 樹

由頁面中所有 HTML 元素所組成

  • 瀏覽器解析時，會將程式碼變成 DOM 樹。
  • document：根節點，表示整份 HTML 文件，存取 HTML 皆由其開始。
  • 節點 node：表示每一物件，也是 DOM 中所有 Object 的父類。
    • 元素節點 element node
    • 文字節點 text node
    • 屬性節點 attribute node
    • 註解節點 comment node

DOM API

可以存取並改變文件架構和內容的屬性方法

查找元素

  • 根據 HTML 標籤

    • document.getElementById(id)

    • document.getElementsByTagName()

    • document.getElementsByClassName()

  • CSS 選擇器

    • document.querySelector()：只選取第一個匹配的元素

    • document.querySelectorAll()：選取所有匹配到的元素

CSS style

  • 行內（不易修改）

    • element.style.cssProperty = newStyle

    • element.style.cssText = newCSS

  • element.classList

    可透過定義的方法修改元素的 class 屬性，比如 add()、remove()、toggle()。

修改元素內容

  • element.innerHTML()：抓取標籤內的所有內容，包含 HTML 標籤並改變。
  • element.innerText()：只抓取標籤內的文字並改變
  • element.outerHTML()（少用）：抓取整個元素，可用來重新設置。

刪除與插入元素

  • element.appendChild()：移除子元素

  • element.removeChild()：插入元素在最後

屬性

  • element.setAttribute()：設定元素的屬性，若已存在則更新。
  • element.getAttribute()：取得當前的屬性
  • element.attributes：取得所有的屬性
