[JS] 事件 Event

2022-03-20

當使用者對網頁進行操作，網頁要根據事件 Event 去產生對應的更新。

Event 對象

事件的狀態，以下為常用的：

事件 描述
click 滑鼠點擊物件時
keydown 按下鍵盤按鍵時
mousedown 按下滑鼠按鍵時
mousemove 滑鼠移動時
mouseout 滑鼠離開指定元素四周時
submit 按下送出按鈕時

事件處理器 Event Handler

行内（不推薦）

<button onclick="bgChange()">Press me</button>

  • 只能在一元素上使用一事件
  • 混用 HTML、JS 難以解析
  • 易被植入惡意程式碼

屬性

事件監聽 addEventListener

監聽到指定的事件發生時，執行對應的任務函式。

  1. 選擇要出現的事件

  2. function 為匿名以帶入要出現的功能

  3. 布林值（預設 false）決定要不要產生冒泡事件，不指定會自動產生。

    • false → 冒泡事件：從指定的元素往外找

    • true → 捕獲事件：從最外層找到指定元素

事件委派機制（傳遞機制）

原則

  • 先捕獲，再冒泡。
  • 當事件傳到 target 本身，沒有分捕獲跟冒泡。

階段

  1. 捕獲階段 Capturing Phase
  2. 傳遞到元素本身 Target Phase
  3. 冒泡階段 Bubbling Phase

冒泡事件 Event Bubbling

由內往外從事件觸發的節點（元素）向根部（document）往上傳遞的過程

  1. button
  2. div
  3. body
  4. html
  5. document

捕獲事件 Event Capturing

由外往內從根部（document）找到事件觸發的節點（元素）往下傳遞的過程

  1. document
  2. html
  3. body
  4. div
  5. button
