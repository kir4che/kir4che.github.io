[React] 組件 Component

2022-03-20

定義

可將 UI 拆分成獨立且重複使用的程式碼

建立

  1. Function

    React 16.8 新增 Hook 功能，關注的是重新 render 後要做什麼。

    • 可用 Hooks
    • 沒有生命週期
    • 沒有 this，但可透過 useState() 使用 state。
    • props 傳遞透過 function 參數傳入

  2. Class

    用 ES6 class 來定義 component，關注的是生命週期。

    • 無法使用 Hooks
    • 有 this 指向，可用 this.state，擁有自己的 private data。
    • props 傳遞透過 this.porps

  3. creactClass（React 16 以後不支援）
