正規表達式 Regular Expression（Regex、RegExp）

確認字串是否符合規則，並構成搜索模式 search pattern。

用途

  • 尋找或取代匹配字串
  • 驗證使用者輸入欄位
  • 擷取某段想要的資訊

特點

  • 簡化對字符串的複雜操作

  • 正則越長，可讀性越差。

建立

字面值

新增 RegExp 物件

規則

驗證字元

  • /./ 任意字元

  • /a/ 字串含 a

  • /a./ 字串含 a＋任一字元

特定字元

  • [A-Z] 大寫英文字母
  • [a-z] 小寫英文字母
  • [0–9] 數字 0～9
  • \d 數字
  • \D 非數字
  • \w 數字字母與底線＝[A-Za-z0–9_]
  • \W 非數字字母與底線＝[A-Za-z0–9_]
  • \s 空白字元
  • \S 非空白字元

開頭結尾

  • ^ 開始　ex. /^em/：開頭是 em
  • $ 結束　ex. /ma$/：結尾是 ma

多個字元

  • [] 驗證括弧內的每一字元　ex. /[abc]/：包含 a 或 b 或 c
  • [^] 反過來　ex. /[^abc]/：不包含 a 或 b 或 c

次數

  • * 0 次以上
  • + 1 次以上
  • ? 0 或 1 次
  • {m} m 次
  • {n,} 最少 n 次
  • {m,n} m～n 次
  • {m,n}? m～n 次之間最少次的
