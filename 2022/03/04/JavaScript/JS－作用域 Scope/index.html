[JS] 作用域 Scope

2022-03-04

規範哪些變數在哪些程式碼中可以取用的規則

作用範圍

全域作用域 Global Scope

函式外宣告

  • 瀏覽器頁面開啟時，自動生成一全域性作用域 window
  • 會一直存在到頁面關閉為止

缺點

  • 允許重複宣告

  • 不支援 Block Scope

  • 不支援常數

影響

  • 命名空間汙染、混亂

  • 易被存取，可能被修改而導致臭蟲。

  • 性能差，須一直追蹤。

區域作用域 Local Scope

  • 函式作用域 Function Scope

    函式 {} 內宣告

  • 區塊作用域 Block Scope（ES6）

    於函式中的 { } 內宣告 ➪ 防止不當存取

規則

靜態、語彙作用域 Static、Lexical Scope

以宣告函式的地方作為作用域鏈，為 JS 所採用。

外層 Scope 無法取用內層變數，但內層 Scope 可以取用外層變數。

動態作用域 Dynamic Scope

以呼叫函式的地方作為作用域鏈

作用域鏈 Scope Chain

記錄每個環境之間切換的關聯 ex. global -> func()

  1. 函式執行時先從內部找變數
  2. 找不到再一層層往外找
  3. 若到全域還找不到，則拋出錯誤。
