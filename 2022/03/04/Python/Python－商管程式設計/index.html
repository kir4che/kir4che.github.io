Scheduling

Makespan minimization 最小化排程

  • Load balancing 負載平衡
  • Heuristic algo. 啟發性演算法、經驗法則：利用經驗來搜尋近似最佳解

Longest processing time first（LPT）

  • For makespan minimization
    1. 降冪排序 jobs 的 processing time
    2. 分配 job 給目前能最早完成時間的機器
  • 也稱作
    • Iterative algo.
    • Greedy algo.：採取當前最好、最佳選

Invetory Control

Common inventory policies

  • the（Q, R）policy：I < R（= reorder point）則訂 Q 個貨 ➪ 最小化存貨

    • optimizing the（Q, R）policy

      • Minimize the sum of inventory, ordering, and shortage cost.

    • optimal R

      • the expected total cost for the future, need to estimate / forecast / predict future demands.
      • 無未來預測 ➪ for the past ? days

  • the（s, S）policy：當你的存貨 I < s 則訂 S

Automatic ordering

  • continuous review system
    • 每經過 POS 售出，請檢查存貨。
    • 達到 reorder point 則訂貨
  • periodic review system
    • 定期檢查存貨
    • 達到 reorder point 則訂貨

並算出最佳訂貨量 q、期望利潤 $\pi(q) = rE[\ min${ q, D }$\ ] − cq$
