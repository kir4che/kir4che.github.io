[DSA] 搜尋 Search

2022-02-13

分類

  • 內部搜尋

    資料量小，可從 RAM 資料檔中尋找資料。

  • 外部搜尋

    資料量較大，通常先分次載入部份資料，再用內部排序後回存或合併結果。

線性搜尋 Linear Search（= 循序搜尋 Sequential Search）

從頭逐一比對各筆資料值與目標，直到找到或搜完為止。

複雜度

  • 時間：O(1) → O(n) → O(n)

  • 空間：O(1)

程式碼

二分搜尋 Binary Search

在已排序陣列 中搜尋某一元素

過程

  1. 排序
  2. 找出中間值
  3. 比對目標與中間值（若大於目標往左找）
  4. 重複 3. 縮小範圍至找到或找完

複雜度

  • 時間：O(1) → O(logn) → O(logn)

  • 空間：O(1)

程式碼

內插搜尋 Interpolation Search

從已排序資料中以斜率預測目標所在位置

複雜度

  • 時間：O(1) → O(n) → O(log(logn))

  • 空間：O(1)

程式碼

指數搜尋 Exponential Search

搜尋無限、無邊界的已排序陣列

過程

  1. 從索引位置 $2^0$ 開始，不斷遞增 $2^1、2^2$ 直到 $2^i$。

  2. 若比目標大，會停止指數增長，改用二分搜尋來尋找。

複雜度

  • 時間：O(1) → O(logn) → O(logn)

  • 空間：O(1)

程式碼

深度優先搜尋 Depth-first Search（DFS）

過程

  1. 選定一起始點走訪
  2. 選與其相鄰且未走過的點，再以此為起始點。
  3. 重複 1～2.，直到走完所有頂點。

圖解

廣度優先搜尋 Breadth-first Search（BFS）

找出兩物之間的最短距離

過程

  1. 選定一起始點走訪
  2. 選與其相鄰且未走過的點
  3. 重複 1～2.
  4. 走完所有與其相鄰的點後，尋找下一層頂點。
