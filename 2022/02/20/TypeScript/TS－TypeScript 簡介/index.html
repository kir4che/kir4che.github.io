[TS] TypeScript 簡介

2022-02-20

簡介

TypeScript 為 JavaScript 的超集

優點

  • 簡化 JS

  • 提高可讀性

  • 可依 ES5、6 標準來編譯

擴展功能

  • 資料型別的檢查
  • 介面 interface、枚舉 enum、Mixin
  • 泛型 generics
  • 命名空間 namespace
  • 元組 tuple

編譯設定

基礎

  • target 指定 JS 版本

  • module 生成哪種模組

  • outFile 輸出全部整合

  • outDir 設定輸入資料夾

  • noEmit 不產生輸出檔案

檢查模式

  • strict 嚴格模式

module 解析選項

  • esModuleInterop 兼容模組導入的方式
