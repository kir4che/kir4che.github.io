useEffect 副作用控制

React 每次更新完 DOM 後觸發 useEffect

useEffect(callback, arr)

Param

  • callback function：定義 componentDidMount／componentDidUpdate 要做的事
  • array：當哪些 state、props 改變時，重新觸發 useEffect。

Rules

  • 宣告於最外層 {}
  • 僅用於 function component、自訂 hook

Use in

  • 手動操作 DOM
  • add／removeEventListenser
  • set／clearInterval
  • 使用外部函式庫
  • 向伺服器請求資料
