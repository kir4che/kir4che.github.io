JS 物件標記法 JavaScript Object Notation（JSON）

一種輕量級的資料交換格式，

  • 只是語法參考 JS 並非只有 JS 可用

優點

  • 便於閱讀、書寫
  • 支持多種語言
  • 比起 XML 格式更小
  • 具有長時間的穩定性（∵ 創始人宣稱此格式永不升級）

缺點

  • 目前沒有 XML 如此通用性

轉換

JSON.parse()

JSON.stringify()
