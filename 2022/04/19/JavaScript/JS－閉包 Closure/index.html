[JS] 閉包 Closure

2022-04-19

簡介

函式中的函式，也是一物件，用於存取外部函數的變數。

💡與語彙作用域 Lexical Scope 密不可分，詳見 [JS] 作用域 Scope。

優點

  • 避免使用全域變數
  • 延伸了變數的作用範圍
  • 減少命名空間之混亂、物件的相依性

缺點

  • 被引用的私有變數不能被銷毀 ➪ 記憶體洩漏

    解決：變數用完後手動賦值 null

  • 跨域存取 ➪ 性能損失

何時不該使用？

  • 作用域過多
  • 不該使用也不要用
  • 記憶體回收 Garbage Collection：不正確使用閉包可能導致記憶體洩漏 Memory Leak
