[JS] JavaScript 是什麼？

2022-07-01

一種物件導向的用戶端腳本語言，能夠動態更新網頁內容、控制多媒體、動畫…。

  • 解釋性：原始程式碼不需編譯，直接在瀏覽器上運行時被解釋。
  • 基於原型的面向物件：能運用自己已創建的物件
  • 事件驅動：在主頁執行某種操作所產生的動作。JS 可直接對用戶端做出回應，並以事件驅動進行，無需經過 Web 服務程式。
  • 跨平台：JS 依賴於瀏覽器本身，與操作環境無關。
  • 安全性：不允許訪問本地磁碟，且不能將數據存入伺服器上，也不允許修改、刪除網路文本，有效防止數據丟失。
