[JS] BOM＆DOM

2022-06-25

Browser Object Model（BOM）

瀏覽器本身（window 物件）所有功能的核心，作為 JS 與瀏覽器的溝通窗口，無關網頁內容。

結構

  • screen：使用者的螢幕畫面資訊
  • location：頁面網址 (URL)
  • history：瀏覽器的上下頁
  • navigator：瀏覽器資訊
  • frames：瀏覽器框架

常見操作

window

  • window.open() 開啟新視窗 ↔ window.close()

對話視窗

  • alert() 警告

  • confirm() 確認

  • prompt() 文字輸入

計時器

一段時間過後，自動執行某個函式 (callback)。

  • setTimeout() 執行一次就停止 ↔ clearTimeout()

  • ``setInterval()一直執行下去 ↔clearInterval()`

history

  • history.back() 回到上一頁 ↔ history.forward()
  • history.go() 跳至第幾頁

Document Object Model（DOM）

提供了一個文件（樹）的結構化表示法，定義程式可存取並改變架構、風格和內容的方法。

document 是 DOM tree 的根節點，表示整份 HTML 文件。

取得 DOM 元素

ID

  • document.getElementById()

Tag

  • document.getElementsByTagName()

Class

  • document.getElementsByClassName()

CSS 選擇器

  • document.querySelector()
  • document.querySelectorAll()

Node

  • node.childNodes() 元素的所有子節點
  • node.children() 元素的所有 Element 節點
  • node.firstChild() 第一個節點 ↔ lastChild()
  • node.previousSibling() 元素的前一個節點 ↔ nextSibling()

節點物件的屬性

  • node.nodeType() 取得該節點類型
  • element.innerHTML／Text() 取得 DOM 元素節點中的內容、改變元素的內容
  • element.outerHTML() 同上但會返回節點本身的內容
  • node.textContent() 第一個節點 ↔ lastChild()
    • textContent 返回 <script>、<style> 的所有內容，但 innerText 不會。
    • innerText 不會返回看不見的元素，但 textContent 會。

節點操作

  • document.createElement() 建立一新元素
  • document.createTextNode() 建立一新文字節點
  • parentNode.appendChild() 新增一新子元素到現有的最後面 ↔ removeChild()
  • parentNode.insertBefore() 新增一新子元素到現有的前面

HTML attr

element.hasAttribute() 檢查元素是否有某個屬性

element.getAttribute() 取得元素的屬性值

element.setAttribute() 新增元素的屬性 ↔ removeAttribute()

element.attributes 取得元素所有的屬性

CSS style

element.style.cssProperty = newStyle

element.style.cssText = newCSS

change 物件內容改變時

error 當圖片或文件下載產生錯誤時
focus 當物件被點擊或取得焦點時
keydown 按下鍵盤按鍵時
keypress 按下並放開鍵盤按鍵後
keyup 按下並放開鍵盤按鍵時
load 網頁或圖片完成下載時
mousedown 按下滑鼠按鍵時
mousemove 介於over跟out間的滑鼠移動行為
mouseout 滑鼠離開某物件四周時
mouseover 滑鼠進入一個元素 (包含進入該元素中的子元素) 四周時
mouseup 放開滑鼠按鍵時
resize 當視窗或框架大小被改變時
scroll 當捲軸被拉動時
select 當文字被選取時
submit 當按下送出按紐時
beforeunload 當使用者關閉 (或離開) 網頁之前
unload 當使用者關閉 (或離開) 網頁之後
