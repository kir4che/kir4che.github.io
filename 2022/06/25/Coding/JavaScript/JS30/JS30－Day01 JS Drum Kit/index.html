[JS30] Day01 JS Drum Kit

2022-06-25

功能

以 JS 使鍵盤按下後播放出對應按鍵的音效

事件監聽

element.addEventListener(event, func, useCapture)

按下按鍵後播放聲音

  • 偵測按下鍵盤事件

    • 偵測 window.addEventListener("keydown", (e) => {...});
    • 按下哪一鍵－取得 keyCode
      • keyCode 查詢：https://keycode.info/

  • 播放聲音

    • 回到音訊開頭 audio.currentTime = 0;
    • 播放 audio.play();

data-*

自訂屬性便於 JS 取得，為全域屬性。

存取

  • element.dataset.keyname
  • element.dataset[keyname]

transitionend

發生於 CSS transition 完成後

transition 轉場

transition: prop duratioin timing-func delay;

transitionEvent（唯讀）

event.propertyName

取得與 transition 有關的屬性名稱

類別屬性

element.classList

  • 新增 add()
  • 移除 remove()
  • 切換 toggle()

Target

  • 當前事件對象 event.currentTarget

  • 觸發事件的元素 event.target

transform

  • 位移 translate(x,y)
  • 縮放 scale(x,y)
  • 傾斜 skew(deg,deg)
  • 轉軸 rotate(deg)
  • 變形矩陣 matrix(a,b,c,d,e,f)
