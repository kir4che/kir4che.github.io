選擇器

基本選擇器

  • 通用 *
  • 元素 elm
  • class .class
  • id #id：具唯一性，常搭配 JS。
  • 屬性[attr]

組合選擇器

  • 後代 .elm1 elm2
  • 子代 .elm1＞elm2
  • 相鄰兄弟 .elm1＋elm2：elm1 後的第一個 elm2
  • 一般兄弟 .elm1～elm2：elm1 後的所有 elm2

偽類選擇器

  • 未訪問過的元素 :link ↔ :visited
  • 滑鼠放在元素上之狀態 :hover
  • 獲得焦點的元素 :focus
  • 點擊元素時 :active
  • 於元素之前插入內容 :before ↔ :after
