2022-06-25

React

JavaScript UI 庫

優點

  • 宣告式 Declarative：以 Virtual DOM 概念進行綁定、渲染
  • 高效率重用[組件](React－組件 Component.md)
  • 與 Backend for Frontend 相性良好
  • 可開發原生 APP ➪ React Native
  • 強大的社群生態系

Virtual DOM：以 JS 物件模擬 DOM 結構而產生的樹狀結構，不直接操作 DOM，而是暫存再更新 ➪ 提升效能

ReactDOM：把 Virtual DOM 渲染至文檔中變成實際瀏覽器的 DOM（= React.js App 進入點）

渲染機制

JSX（選用）

JS 語法擴展，也是一種語法糖 Syntatic Sugar。

語法糖：此語法對語言的功能無影響，但便於程式設計師使用。

限制

  • 只能有一個根元素 root
  • 駝峰式命名
  • 所有元素都須閉合
  • 保留字
    • class → className
    • for → htmlFor
  • 以 { } 篩寫 JS 程式碼
    • 行內樣式
    • 表達式
    • 屬性 attr
    • 綁定事件
    • 疊代元素

優點

  • 更加語意化且易懂的標籤
  • 實現宣告式 Declarative（注重 what to），而非命令式 Imperative（注重 how to）之寫法 ➪ 關注點分離
  • 結合原生 JS 語法

Node Package Manager（NPM）

Node.js 下的主流套件管理工具

Babel

將 ES6+ 、JSX 等程式碼轉換成不支援的瀏覽器能看懂的語法
