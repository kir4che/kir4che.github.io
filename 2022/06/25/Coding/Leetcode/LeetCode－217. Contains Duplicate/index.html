217. Contains Duplicate

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

Example

Input: nums = [1,2,3,1]

Output: true

Input: nums = [1,2,3,4]

Output: false

Solution 1（22ms）

一開始想到先排序陣列，再找相鄰數字是否重複。

Solution 2（6ms）

使用無序且元素不可重複的 HashSet，若裡面已有某個元素，再次新增時會返回 null。
