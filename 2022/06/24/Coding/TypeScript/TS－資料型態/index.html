指定型別

Type Inference 型別推斷

在沒有明確指定型別時推測出型別，編譯時會報錯。

Type Annotation 型別註記

變數名稱: 型別＝值

Type Assertions 型別斷言

＜型別＞= 值

值 as 型別

💡並非轉換，不可斷言成聯合型別中不存在的型別。

原始資料型別

string

number

boolean

空值

  • null：賦予空值

  • undefined：宣告卻未被賦值

物件資料型別

object

arrays

  • 定義

class 類別

擴充型別

tuple 元組

tuple 就是合併了不同型別的物件。

enum 列舉

管理一系列的常數，作為狀態判斷用。

interface 介面

Literal Type 字面值型別

某些特殊的「值」可當作「型別」使用。

  • string literal types

  • numeric literal types

  • non-literal types　ex. boolean、enum、interface, etc.

特殊型別

  • any 任意值

    可被賦值為任意型別，宣告時未指定型別也是 any。

  • unknow

    接受任何形態賦值，只能賦值給自己或 any。

  • never

    不應該存在的型別，用於錯誤處理函式，且需回傳，但因錯誤導致不會有回傳值。

複合型別

  • union types 聯合型別

    以｜取多種型別的其一 ≒ OR

    Type Alias 型別別名

    給一型別起個新名字

  • intersection types 交集型別

    以＆取多種型別的其一 ≒ AND

Generic 泛型

定義函式、介面或類別時先不指定型別，使用時再指定。
