[React] CSS in JS

2022-06-24

一個 CSS in JS 函式庫

用途

  • 獨立 Class ➪ 避免重複讀取而互相影響
  • 支持組件間繼承 ➪ 可維護性 ↗
  • 能以 Props 控制 CSS

優點

  • 提取可重複使用的樣式進行管理
  • 不用想 className
  • 避免沒用的 CSS 樣式堆積
  • 比較好找對應元件

缺點

  • 生成的 className 是隨機字串 ➪ 除錯不易
  • 預設無 CSS 語法高亮、提示

安裝

  • npm install --save styled-components
  • yarn add styled-components
