[Tailwind] CSS 筆記整理

2022-06-24

安裝＆設定

yarn add postcss-cli autoprefixer -D

產生配置檔

yarn tailwindcss init -p

tailwind.css

編譯

  • npx tailwindcss -i tailwind.css -o style.css
  • module.exports = { purge: [ "./*.{html,js}", "./**/*.{html,js}", ], ... }npm run build

Purge 設定

於 tailwind.config.js 更改 ➪ 環境優化

  • 檢查檔案

  • 安全名單 Safe List

    讓 Tailwind 移除樣式時略過他們

    💡若把 safelist 寫在 purge，則只接受字串格式，不接受正規表示式 regex。

  • Just In Time（JIT）

    隨時監控 purge 設定的監聽範圍中的文件

    💡即時編譯 Just-In-Time Compilation：不斷實時分析正在執行的程式碼，並認定程式中的某些內容，更新時就只需要部份編譯。

    • 啟用
    • 功能

      • 任意使用 Variants

        <input class="active:bg-indigo-200">

      • 任意值 class

        <button class="bg-[#00b900] w-[40px] h-[20px]">加入</button>

      • !important

        <div class="sm:hover:!font-bold">

  • 監控伺服器 Watch Server
