[JS] jQuery 筆記

2022-06-24

jQuery 庫

JavaScript 函數庫，為事件處理特別設計的。

特性

  • HTML 元素選取、操作
  • CSS 操作
  • HTML 事件函數
  • JS 特效、動畫
  • HTML DOM 遍歷、修改
  • AJAX
  • Utilities

引用

  • 下載 jQuery
  • <script src="jquery.js"></script>

語法

$(選擇器).行為()

💡若同時用 jQuery 及其他框架，而都以 $ 作為簡寫怎麼辦？ noConflict()

特效

協助快速控制 CSS

  • show() 顯示 ↔ hide()
  • toggle() 切換隱藏／顯示
  • 淡入／出
    • fadeIn(speed)
    • fadeOut(speed)
    • fadeToggle(speed)
    • fadeTo(speed, opacity)
  • 滑入／出
    • slideDown(speed, callback)
    • slideUp(speed, callback)
    • slideToggle(speed, callback)
  • delay(speed) 延遲時間
  • stop() 停止正在發生的特效
  • animate(css, speed) 應用自定義動畫

Callback：於當前動畫完成後調用

Chaining：把多個動作／方法連接起来

HTML

内容＆屬性

  • text() 設置／回傳内容
  • html() 同上（包括 HTML 標記）
  • val() 設置／回傳表單值
  • attr() 設置／改變屬性值

新增内容

  • append() 結尾
  • prepend() 開頭
  • after() 元素後
  • before() 元素前

刪除内容

  • remove() 刪除包括其子元素
  • empty() 只刪除子元素

CSS

  • addClass() 新增一或多個類

  • removeClass() 刪除一或多個類

  • toggleClass() 切換新增／刪除

  • css() 設置／回傳元素之屬性

attr() 與 css() 之差別

  • attr() → HTML
  • css() → style

遍歷

祖先

向上遍歷 DOM 樹

  • parent() 回傳父元素
  • parents() 回傳所有祖先元素
  • parentsUntil() 回傳兩元素之間的所有祖先元素

後代

向下遍歷 DOM 樹

  • children() 回傳子元素
  • find() 回傳所有後代元素

同胞

擁有相同的父元素，水平遍歷 DOM 樹。

  • siblings() 回傳所有同胞元素

  • 向前

    • next() 回傳下一同胞元素
    • nextAll() 回傳所有跟隨的同胞元素
    • nextUntil() 回傳兩參數之間所有跟隨的同胞元素

  • 向後

    • prev()
    • prevAll()
    • prevUntil()

過濾

  • first() 回傳首個元素 ↔ last()
  • eq() 回傳帶有指定索引的元素
  • filter() 回傳匹配標準的元素，否則刪除 ↔ not()
