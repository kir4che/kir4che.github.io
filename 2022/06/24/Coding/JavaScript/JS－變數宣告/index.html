[JS] 變數宣告

2022-06-24

Scope 作用域

Global Scope

函式外宣告

缺點

  • 允許重複宣告
  • 不支援 Block Scope
  • 不支援常數

影響

  • 命名空間汙染、混亂
  • 易被存取，可能被修改而導致臭蟲。
  • 性能差，須一直追蹤。

Function Scope

函式 {} 內宣告

Block Scope

於函式中的 { } 內宣告 ➪ 防止不當存取

宣告

var

in 函式作用域

let

in 區塊作用域

const

宣告時必須賦值 in 區塊作用域

💡let、const 同區塊內禁止重複命名、不會洩漏到全域
