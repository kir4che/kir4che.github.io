[DSA] 排序 Sorting

2022-06-24

分類

  • 內部排序 Internal sort（= 陣列排序）

    資料量少，於 RAM 就能完成，一般演算法皆為內部排序。

  • 外部排序 External sort（= 檔案排序）

    資料量較大，通常先分次載入部份資料至記憶體，再用內部排序後回存或合併結果。

相同值排序前後之順序

  • 穩定排序法 Stable sorting ➪ 相同

排序效率

  • 簡單（初等）排序：排序演算法簡單，但執行時間較長 $O(n^2)$ ↔ 高等排序 $O(nlog^2n)$

常見排序

選擇排序 Selection sort

每次把沒排過中的 min 丟到前面

動畫

程式碼

插入排序 Insertion sort

每一未排序資料由右而左來找位置插入

動畫

程式碼

氣泡排序 Bubble sort（= 交換排序）

依序比較相鄰兩個元素，右 > 左則交換。

動畫

程式碼

堆積排序 Heap sort

步驟

  1. 陣列 → 最大堆積 Max Heap（遞增）或最小堆積 Min Heap（遞減）
  2. 再以完全二元樹來排序

動畫

程式碼

合併排序 Merge sort

採用分治法 Divide-and-Conquer，陣列拆分後再排序合併。

動畫

程式碼

快速排序 Quick sort

採用分治法 Divide-and-Conquer

步驟

  1. 選出一元素當基準 pivot

    左邊的元素都＜pivot、右邊的元素都 $\ge$ pivot（不用排序）

  2. 對左右邊的 array 重複 1.

動畫

程式碼

希爾排序 Shell sort（= 增量遞減排序 Diminishing Increment sort）

改良於插入排序

  • 將資料透過不同的間距 gap 拆分
  • 每段資料各自做插入排序
  • 利用每次排序出來的結果來加速下一次排序

動畫

程式碼

計數排序 Counting sort

先計算每個元素出現於序列中的次數，再用其進行排序。

動畫

程式碼

桶排序 Bucket sort

每一桶子對應資料區間，先將資料分配到不同桶，內部再各自排序。

速度

  • 當資料均勻分配每一桶時最快
  • 都分配同一桶中最慢

動畫

程式碼

基數排序 Radix sort

根據某個基數對元素進行分配排序

類型

  • Least Significant Digital（LSD）：由右到左
    1. LSD of each key：取得每一資料鍵值的最小位數（LSD）
    2. Sorting subroutine：依該位數大小排序
    3. Repeating：取得下一有效位數，並重複 2. 至最大位數（MSD）為止
  • Most Significant Digital（MSD）：由左到右
