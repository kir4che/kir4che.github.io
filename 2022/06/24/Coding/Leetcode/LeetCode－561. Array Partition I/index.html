561. Array Partition I

Given an integer array of integers, group these integers into pairs such that the sum of for all is maximized. Return the maximized sum.

Example

Input: nums = [6,2,6,5,1,2]

Output: 9

Explanation: min(2, 1) + min(2, 5) + min(6, 6) = 1 + 2 + 6 = 9

Solution（10ms）

先排序再兩兩分成一組，每組左邊為 min，求左邊之和。
