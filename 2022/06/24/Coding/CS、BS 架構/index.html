CS、BS 架構

2022-06-24

Client／Server 主從式架構

透過 HTTP 協定送出請求 request 給 server 提取 get，當 server 接收到回應 response 後開始處理，並回傳一個回應 response 給 client。

➪ client、server 之間透過 HTTP 協定進行溝通

  • 交互性強
  • 具有安全的存取模式
  • 響應速度快
  • 適於處理大量數據
  • 較難以維護、管理（成本高）

Server

提供 web 服務相關的伺服器

Client（主動）

任何可以呈現 server 端所傳來的資訊，並提供使用者本地服務的程式。

Browser／Server 瀏覽器-伺服器

只需要瀏覽器即可與 web 伺服器互動

  • 分布性強
  • 易於維護（成本低）
  • 開發簡單
  • 共享性強
