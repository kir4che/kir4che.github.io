[Java] Swing 視窗設計

2022-06-24

Swing

採用 Model-View-Controller 設計，提供各種 GUI 元件。

建立視窗

  1. 視窗初始化 Jframe()

  2. 內容面板

    • JPanel

      1. 元件之容器 JPanel()：集中元件
      2. 視窗設定內容面板 frame.setContentPane(contentPane)
      3. 添加元件 panel.add(component)

    • 設定佈局 frame.setLayout(LayotManager mgr)

      類型

      • FlowLayout：一行行排
      • BorderLayout：分為東西南北中
      • CardLayout：每個元件都是卡片
      • GridLayout：平均劃分為表格
      • GridBagLayout：加上合併儲存格的 GridLayout
      • GroupLayout：分層分組元件
      • null 不佈局：自己設座標

      💡可不寫 JPanel，會直接調用至 ContentPane，若有寫則改為 panel.setLayout(LayotManager mgr)。
