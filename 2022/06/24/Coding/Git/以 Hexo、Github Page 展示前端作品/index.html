專案上傳

  1. git init
  2. git add .
  3. git commit -m “first commit”
  4. git branch -M main
  5. git remote add origin git@github.com:kir4che/項目名稱.git
  6. git push -u origin main

Github Page

  1. yarn add –dev gh-pages

  2. 新增 gh-pages 分支

  3. package.json

  4. yarn run deploy
